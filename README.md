📝 Task Manager

A modern Task Manager built using React and Vite.

✨ Features

- ➕ Add Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- ✅ Mark Tasks as Complete/Undo
- 🔍 Search Tasks
- 📂 Filter Tasks (All / Completed / Pending)
- 📊 Task Statistics
- 💾 Local Storage Support

## 🛠️ Technologies Used

- React
- Vite
- JavaScript
- CSS

## 🚀 How to Run

```bash
npm install
npm run dev
```
